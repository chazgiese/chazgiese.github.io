RemindPlse

RemindPlse is a mobile app that politely reminds you to do things. Release date for iOS is set for late december.

My Role
  • Product Designer
  • Software Engineer

Problems

  • Current apps are overly complicated
  • Too many steps for creating a time based reminder
  • UI clutter

Goals

  • Singular focus on reminders
  • As few steps as possible to create a new reminder
  • Distill UI to the core components
  • Encourage users to make more reminders because it's so easy

Icons

I designed 3 icons to represent: Today, Reminders, and Categories.
